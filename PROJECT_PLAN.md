# CodeGuardian AI

## Overview

CodeGuardian AI is an AI-powered code review assistant that helps developers review GitHub Pull Requests automatically. It analyzes code changes using Large Language Models (LLMs) and provides intelligent suggestions to improve code quality, security, performance, and maintainability.

The project aims to reduce manual code review effort while helping developers write cleaner and more reliable code.

---

# Problem Statement

Manual code reviews are time-consuming and may miss bugs, security vulnerabilities, performance issues, or coding standard violations. Developers need an intelligent assistant that can review Pull Requests instantly and provide meaningful suggestions before human review.

---

# Objectives

* Improve code quality through intelligent suggestions.

# Project Workflow

1. User enters GitHub repository information.
2. Application fetches Pull Request details.
3. Changed files are retrieved.
4. AI analyzes the code changes.
5. Suggestions are categorized and displayed.
6. A review report is generated for the user.
